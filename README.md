# Инфраструктурный контур — сайт

Корпоративный сайт ООО «Инфраструктурный контур»: технологическое присоединение, инженерная инфраструктура, ЖКХ, согласования (Москва и Московская область).

## Стек

- [Next.js](https://nextjs.org/) 15 (App Router)
- React 18, TypeScript

## Запуск локально

```bash
npm install
npm run dev
```

Сайт: http://localhost:3000

## Сборка

```bash
npm run build
npm start
```

## Деплой

Проект развёрнут на Vercel: https://trastenergy-site.vercel.app

## Структура

- `app/` — страницы (маршруты)
- `components/` — UI-компоненты
- `lib/data/` — контент услуг, нормативов, калькуляторов
- `public/` — статика и бренд
